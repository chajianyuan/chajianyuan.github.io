二、post 提交数据方式

1、application/json

这是最常见的 json 格式

2、application/x-www-form-urlencoded

浏览器的原生 form 表单，如果不设置 enctype 属性，那么最终就会以 application/x-www-form-urlencoded 方式提交

3、multipart/form-data

当你需要提交文件、非 ASCII 码的数据或者是二进制流数据，则使用这种提交方式。

4、text/xml

这种直接传的 xml 格式
