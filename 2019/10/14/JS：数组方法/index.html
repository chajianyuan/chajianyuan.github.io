JS：数组方法

发表于 2019-10-14 更新于 2020-02-24 分类于 技术

一、改变原数组

1、push（添加元素）

向数组后面添加元素，返回值为添加完后的数组长度

2、pop（删除元素）

从后面删除数组元素，只能删除一个，返回被删除的元素

3、shift（删除元素）

从前面删除数组元素，只能删除一个，返回被删除的元素

4、unshift（添加元素）

从前面添加数组元素，返回添加后的数组的长度

5、splice（添加、删除、替换元素）

（1）添加元素

返回空数组

array.splice(index,0,item...)

（2）删除元素

返回被删除的元素

array.splice(index,howMany)

（3）替换数组

返回要被替换的元素

array.splice(index,homMany,item...)

6、sort

对数组进行排序，返回排好序的数组

7、reverse

将数组反转，返回反转后的数组

8、copyWithin（替换元素）

替换数组中的元素，返回替换后的数组

array.copyWithin(复制到指定目标索引，元素复制的起始位置，停止复印的索引位置)

9、fill（替换元素）

用一个固定值填充一个数组中从索引到终止索引（不包括）内的全部元素，返回替换后的数组

array.fill(value[,start,[,end]])

二、不改变原数组

1、concat（拼接数组）

返回连接后的新数组

2、split

将字符串转换成数组

3、slice（截取一部分元素）

切去索引值 start 到索引值 end 的值，不包含 end 索引的值，返回被切掉的数组

4、includes

判断数组中是否包含某个值，返回 true 或 false

5、join

将数组元素拼接起来

6、toString

将数组转换成字符串

7、indexOf

从前往后查找元素索引

8、lastIndexOf

从后往前查找元素索引

三、遍历数组

1、forEach

遍历数组，无 return

2、map

遍历数组，对数组中元素进行修改，返回修改后的数组

3、filter

可以筛选数组中满足条件的元素

4、reduce

接收一个函数作为累加器，接收两个参数（回调函数，初始值），返回一个值
