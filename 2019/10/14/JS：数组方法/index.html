JS：数组方法

一、改变原数组

1、push（添加元素）

向数组后面添加元素，返回值为添加完后的数组长度

let array = [1,2,3,4,5]
array.push(6)   =>  6
array           =>  [1, 2, 3, 4, 5, 6]

2、pop（删除元素）

从后面删除数组元素，只能删除一个，返回被删除的元素

let array = [1,2,3,4,5]
array.pop()  =>  5
array        =>  [1, 2, 3, 4]

3、shift（删除元素）

从前面删除数组元素，只能删除一个，返回被删除的元素

let array = [1,2,3,4,5]
array.shift()  =>  1
array          =>  [2, 3, 4, 5]

4、unshift（添加元素）

从前面添加数组元素，返回添加后的数组的长度

let array = [1,2,3,4,5]
array.unshift(6)  =>  6
array             =>  [6, 1, 2, 3, 4, 5]

5、splice（添加、删除、替换元素）

（1）添加元素

返回空数组

array.splice(index,0,item...)

let array = [1,2,3,4,5]
array.splice(2,0,6)  =>  []
array                =>  [1, 2, 6, 3, 4, 5]

（2）删除元素

返回被删除的元素

array.splice(index,howMany)

let array = [1,2,3,4,5]
array.splice(1,2)  =>  [2, 3]
array              =>  [1, 4, 5]

（3）替换数组

返回要被替换的元素

array.splice(index,homMany,item...)

let array = [1,2,3,4,5]
array.splice(2,1,6)  =>  [3]
array                =>  [1, 2, 6, 4, 5]

6、sort

对数组进行排序，返回排好序的数组

let nums = [3,1,4,6,2,5]
nums.sort()  =>  [1, 2, 3, 4, 5, 6]
nums         =>  [1, 2, 3, 4, 5, 6]

7、reverse

将数组反转，返回反转后的数组

let nums = [3,1,4,6,2,5]
array.reverse()  =>  [5, 4, 3, 2, 1]
array            =>  [5, 4, 3, 2, 1]

8、copyWithin（替换元素）

替换数组中的元素，返回替换后的数组

array.copyWithin(复制到指定目标索引，元素复制的起始位置，停止复印的索引位置)

let array = [1,2,3,4,5]
array.copyWithin(1,3,4)  =>  [1, 4, 3, 4, 5]
array                    =>  [1, 4, 3, 4, 5]

9、fill（替换元素）

用一个固定值填充一个数组中从索引到终止索引（不包括）内的全部元素，返回替换后的数组

array.fill(value[,start,[,end]])

array = [1,2,3,4,5]
array.fill(6,1,3)  =>  [1, 6, 6, 4, 5]
array              =>  [1, 6, 6, 4, 5]

二、不改变原数组

1、concat（拼接数组）

返回连接后的新数组

let array = [1,2,3,4,5]
array.concat([1,2])  =>  [1, 2, 3, 4, 5, 1, 2]
array                =>  [1, 2, 3, 4, 5]

2、split

将字符串转换成数组

let str = '12345'
str.split('')  =>  ["1", "2", "3", "4", "5"]

3、slice（截取一部分元素）

切去索引值 start 到索引值 end 的值，不包含 end 索引的值，返回被切掉的数组

let array = [1,2,3,4,5]
array.slice(1,3)  =>  [2, 3]
array             =>  [1, 2, 3, 4, 5]

4、includes

判断数组中是否包含某个值，返回 true 或 false

array = [1,2,3,4,5]
array.includes(2)  =>  true

5、join

将数组元素拼接起来

array = [1,2,3,4,5]
array.join('')  =>  "12345"

6、toString

将数组转换成字符串

array = [1,2,3,{a:4}]
array.toString()  =>  "1,2,3,[object Object]"

7、indexOf

从前往后查找元素索引

array = [1,2,3,4,5]
array.indexOf(2)  =>  1
array.indexOf(6)  =>  -1

8、lastIndexOf

从后往前查找元素索引

array = [1,2,3,4,5]
array.lastIndexOf(2)  =>  1
array.lastIndexOf(6)  =>  -1

三、遍历数组

1、forEach

遍历数组，无 return

let array = [1,2,3,4,5]
array.forEach(item=>{
	item = item * 2
})
array  =>  [1, 2, 3, 4, 5]

2、map

遍历数组，对数组中元素进行修改，返回修改后的数组

let array = [1,2,3,4,5]
array.map(item=>item*=2)  =>  [2, 4, 6, 8, 10]
array                     =>  [1, 2, 3, 4, 5]

3、filter

可以筛选数组中满足条件的元素

array = [1,2,3,4,5]
array.filter(item=>item>4)  =>  [5]
array  =>  [1,2,3,4,5]

4、reduce

接收一个函数作为累加器，接收两个参数（回调函数，初始值），返回一个值

array = [1,2,3,4,5]
array.reduce((acc,current)=>acc+current,0)  =>  15
array  =>  [1,2,3,4,5]
